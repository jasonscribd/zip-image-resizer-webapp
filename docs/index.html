ZIP Image Resizer

Team utility to resize every image in a ZIP file to 300px width while preserving original proportions.

Open Local App View GitHub Repo

How coworkers can run it

  1. Install Node.js 18+.
  2. Clone the repository.
  3. Install dependencies and start the app.
  4. Open http://localhost:3000 in a browser.
git clone https://github.com/jasonscribd/zip-image-resizer-webapp.git
cd zip-image-resizer-webapp
npm install
npm start

Supported formats: jpg, jpeg, png, webp, tiff, gif, avif.
